A ride-hailing application designed to simulate and manage booking, matching, and tracking of rides between passengers and drivers. This project demonstrates scalable system design, distributed architecture concepts, and practical application of computer science fundamentals.

Features
User Management: Register and authenticate passengers and drivers.

Ride Booking: Request rides with pickup and drop-off locations.

Driver Matching: Match passengers with available drivers in real time.

Trip Tracking: Monitor ride progress and status updates.

Data Storage: Persist user and trip data using relational databases.

Scalable Architecture: Designed with modular components for future cloud deployment.
